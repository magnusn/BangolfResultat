Välja utseende och känsla

Genom alternativet Välja utseende och känsla i menyn Fönster kan man ändra hur hela den grafiska miljön beter sig. Standardalternativet är Java men det går även att välja Naturlig som är anpassad efter operativsystemet som programmet körs på. Ändringen av utseende och känsla kommer inte att märkas förrän BangolfResultat startas om.

Nedan visas hur fönstret för att välja utseende och känsla kan se ut beroende på vilket alternativ som har valts.
