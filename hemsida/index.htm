BangolfResultat


Vad är BangolfResultat? Ett program för att hantera inmatning av bangolfresultat ifrån en tävling. Dessa tävlingar kan sedan sparas som webbsidor och programmet kan även skapa en snittlista för tävlingarna om så önskas. För att få reda på mer om vad programmet kan göra rekommenderas att läsa manualen.

Den senaste versionen finns tillgänglig här nedan som installationsfil för Windows eller ZIP-fil för Mac och Linux. Tänk på att programmet är utvecklat i Java så för att köra det krävs Java Runtime. Installera det genom att klicka på någon av följande länkar om det inte redan är installerat:

  • Automatisk installation av Java.
  • Manuell installation av Java.

OBS! För säkerhets skull rekommenderas det att man säkerhetskopierar systeminställningarna innan en uppdatering av programmet utförs.

Utvecklingen av programmet

Jag som har utvecklat programmet heter Magnus Hovén och är civilingenjör i datateknik. De programmeringskunskaper som jag fick i min utbildning på Lunds Tekniska Högskola har använts för att kunna utföra detta projekt. Tid till projektet fick jag i och med att det inte blev något sommarjobb 2004. Idén till programmet växte fram efter hand. En stor anledning är att jag, medan jag var aktiv i Vinslövs BGK, hanterade en hel del inmatningar av resultat ifrån tävlingar för att kunna publicera dessa på Internet. Då jag ville effektivisera detta arbete så byggde jag upp ett program där man lätt skulle kunna söka upp spelare som tävlar då dessa ofta återkommer och det känns onödigt att mata in hela namnet varje gång.
